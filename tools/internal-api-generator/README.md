# Internal API Generator

## Goal

Generate internal API scaffolding from natural language prompts.

## Example Prompt

`Create API for user notifications`

## Responsibilities

- Create route files.
- Create controller and service files.
- Update OpenAPI specification.
- Create basic API tests.

## MVP Checklist

- [ ] Prompt parser for endpoint name, verb, and resource.
- [ ] API template generator.
- [ ] OpenAPI updater.
- [ ] Test scaffold generator.
